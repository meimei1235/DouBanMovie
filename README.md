## 本项目是一个模仿豆瓣电影的微信小程序，包括首页，搜索，我的三大部分，其中首页包括正在热映，即将上映，新片榜等功能，点击任一个可跳转到电影列表，点击可进入电影详情。
### 之前是调用豆瓣电影的接口，可现在豆瓣电影接口拒绝微信小程序的调用。
